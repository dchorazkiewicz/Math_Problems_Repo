# Task 1 — Coin Tossing

Consider an experiment consisting of tossing a fair coin where the order of outcomes matters.

## 1. Sample space for one coin toss

Sample space:

Ω₁ = {H, T}

Number of elementary outcomes:

|Ω₁| = 2


## 2. Sample space for two coin tosses

Sample space:

Ω₂ = {(H,H), (H,T), (T,H), (T,T)}

Number of elementary outcomes:

|Ω₂| = 4


## 3. Sample space for three coin tosses

Sample space:

Ω₃ = {
(H,H,H),
(H,H,T),
(H,T,H),
(H,T,T),
(T,H,H),
(T,H,T),
(T,T,H),
(T,T,T)
}

Number of elementary outcomes:

|Ω₃| = 8


## 4. Interpretation of an elementary outcome

An elementary outcome represents one specific ordered sequence of results of the coin toss experiment.

Examples:

- In Ω₁ an outcome **H** means the coin landed Heads.
- In Ω₂ an outcome **(H,T)** means the first toss is Heads and the second toss is Tails.
- In Ω₃ an outcome **(T,H,H)** means the first toss is Tails and the next two tosses are Heads.

Each ordered sequence represents exactly one possible result of the experiment.
