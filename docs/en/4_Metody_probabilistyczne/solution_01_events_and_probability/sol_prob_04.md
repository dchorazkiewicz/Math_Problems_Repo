## Task 4

### Idea

The system consists of two elements connected in **parallel**.

In a parallel system the current flows if **at least one component works**.

This corresponds to the **union of events**.

### Solution

Let

A₁ – element a₁ works  
A₂ – element a₂ works

Given:

P(A₁) = p  
P(A₂) = p  
P(A₁ ∩ A₂) = p²

The system works if at least one element works:

P(A₁ ∪ A₂)

Using the union formula:

P(A₁ ∪ A₂) = P(A₁) + P(A₂) − P(A₁ ∩ A₂)

Substitute:

P = p + p − p²

### Result

P = 2p − p²
