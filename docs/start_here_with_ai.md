# Start Here with AI

Download or share the single onboarding document:

[Open `START_HERE_WITH_AI.md`](https://github.com/dchorazkiewicz/Math_Problems_Repo/blob/main/START_HERE_WITH_AI.md)

It gives an AI assistant the complete orientation needed to help a student begin: repository roles, the PDF and LaTeX sources, required note structure, weekly workflow, commit process, and the next action for common starting situations.

You can upload that Markdown file to an AI chat and ask:

> Read this onboarding document. Ask me which week and problem I am working on, then tell me the next concrete step. Follow the linked course instructions and work on only one problem at a time.
