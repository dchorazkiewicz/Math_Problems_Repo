# PROBLEM 5 — FROM FREQUENCIES TO PROBABILITY (STEP-BY-STEP)

Given 1000 throws:

n(1)=168, n(2)=154, n(3)=181, n(4)=167, n(5)=160, n(6)=170


============================================================
## PART A — COMPUTE FREQUENCIES

Rule:
f(A) = n(A) / 1000


------------------------------------------------------------
1) A = {2,4,6}

n(A) = n(2) + n(4) + n(6)
     = 154 + 167 + 170
     = 491

f(A) = 491 / 1000 = 0.491


------------------------------------------------------------
2) B = {1,2,3}

n(B) = 168 + 154 + 181 = 503

f(B) = 503 / 1000 = 0.503


------------------------------------------------------------
3) C = {5,6}

n(C) = 160 + 170 = 330

f(C) = 330 / 1000 = 0.330


------------------------------------------------------------
4) D = {1,3,5}

n(D) = 168 + 181 + 160 = 509

f(D) = 509 / 1000 = 0.509


------------------------------------------------------------
5) E = {1,2,3,4}

n(E) = 168 + 154 + 181 + 167 = 670

f(E) = 670 / 1000 = 0.670


============================================================
## PART B — VERIFY RELATIONSHIPS

------------------------------------------------------------
1) f({2,4,6}) = f(2)+f(4)+f(6)

Left:
f(A)=491/1000

Right:
154/1000 + 167/1000 + 170/1000
= 491/1000

✔ TRUE (disjoint union)


------------------------------------------------------------
2) f({1,2,3,4}) = f({1,2}) + f({3,4})

Compute:

f(1,2) = (168+154)/1000 = 322/1000
f(3,4) = (181+167)/1000 = 348/1000

Sum = 670/1000

Left:
f(E)=670/1000

✔ TRUE (disjoint split)


------------------------------------------------------------
3) f({1,3,5}) + f({2,4,6}) = 1

D = 509/1000
A = 491/1000

Sum = 1000/1000 = 1

✔ TRUE (partition of Ω)


------------------------------------------------------------
4) f({5,6}) = 1 - f({1,2,3,4})

Left:
f({5,6}) = 330/1000

Right:
1 - 670/1000 = 330/1000

✔ TRUE (complement)


============================================================
## PART C — WHEN ADDITION FAILS

------------------------------------------------------------
1) {1,2} ∪ {5,6}

{1,2} and {5,6} are disjoint

f(1,2)=322/1000
f(5,6)=330/1000

Sum = 652/1000

Union:
322 + 330 = 652/1000

✔ WORKS (disjoint sets)


------------------------------------------------------------
2) M = {1,2,3}, N = {3,4,5}

f(M)= (168+154+181)/1000 = 503/1000
f(N)= (181+167+160)/1000 = 508/1000

Sum:
= 1011/1000


Union M∪N = {1,2,3,4,5}
n = 168+154+181+167+160 = 830
f(M∪N)=830/1000


------------------------------------------------------------
3) Why inequality?

Because overlap:

M ∩ N = {3}

Outcome 3 is counted twice in:
f(M) + f(N)


------------------------------------------------------------
4) Double-counted outcome:
→ {3}


============================================================
## PART D — COVERING Ω

------------------------------------------------------------
1) Sum of singletons:

(168+154+181+167+160+170)/1000 = 1000/1000 = 1


------------------------------------------------------------
2) Why = 1?

Because singletons partition Ω:
no overlap, full coverage


------------------------------------------------------------
3) {1,2}, {3,4}, {5,6}

322 + 348 + 330 = 1000
→ 1


------------------------------------------------------------
4) Why again 1?

Still a partition of Ω into disjoint blocks


------------------------------------------------------------
5) GENERAL STATEMENT:

If {A₁, A₂, ..., Aₖ} is a partition of Ω:
(disjoint + covers all outcomes)

then:
Σ f(Aᵢ) = 1


============================================================
## PART E — FROM FREQUENCY TO PROBABILITY

We define probability P(A) such that:
```
1) 0 ≤ P(A) ≤ 1
2) P(∅) = 0
3) P(Ω) = 1
4) If A ∩ B = ∅ → P(A ∪ B) = P(A) + P(B)
5) P(Aᶜ) = 1 - P(A)

```
============================================================
## PART F — CONCLUSION

1) ELEMENTARY OUTCOMES:
→ single results like {1},{2},...,{6}

2) FREQUENCIES:
→ empirical counts from experiment
→ approximate stability emerges with repetition

3) PROBABILITY:
→ abstract limit model of frequencies
→ independent of one experiment
→ defined by axioms ensuring consistency

FINAL IDEA:
frequency → pattern → abstraction (probability space)
