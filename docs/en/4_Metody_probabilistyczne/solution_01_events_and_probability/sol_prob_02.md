## Task 2

### Idea

In reliability systems:

- series connection → intersection
- parallel connection → union

### Solution

Element a1 is connected in series with a block containing elements a2 and a3 connected in parallel.

For the current to flow through the circuit:

- element a1 must work
- at least one of a2 or a3 must work

Let Ai denote the event that element ai is functional.

Therefore the event describing the flow of current is

A1 ∩ (A2 ∪ A3)

### Result

The current flows if event A1 ∩ (A2 ∪ A3) occurs.
