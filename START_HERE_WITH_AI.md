# Basic Mathematics — Course Map for Students and AI

This is the single entry point to the **Basic Mathematics** course materials. A student may read it, upload it to an AI chat, or send the AI this link:

<https://github.com/dchorazkiewicz/Math_Problems_Repo/blob/main/START_HERE_WITH_AI.md>

The purpose of this document is to show where authoritative information is located. An AI assistant should follow the links below instead of guessing or relying only on this summary.

## Course in brief

The course teaches students to read mathematical notation, understand basic mathematical objects, construct justified solutions, present their work, receive feedback, and improve it.

The general rhythm is:

1. attend the lecture;
2. return to the corresponding PDF chapter after the lecture;
3. prepare the complete assigned problem set, working carefully on one problem at a time;
4. present and discuss solutions during exercise classes;
5. listen to feedback given across the group;
6. revise incomplete work in later commits.

AI may support understanding, checking, and mathematical writing, but the student remains responsible for every submitted sentence and formula.

## This document is not a timetable

The course contains twelve thematic units within a fifteen-week semester, but this file does not know the current week, holidays, changes, deadlines, or what the instructor has already assigned. An AI assistant must not guess the current schedule. The student must provide current information announced by the instructor.

## Minimum expectations

For the topic currently assigned by the instructor, the student should read the lecture material, prepare the complete assigned set in separate Markdown notes, understand the reasoning, commit reviewed work, and be ready to present it.

Each finished note must contain the problem statement, given information and goal, method, justified solution, verification where appropriate, and a clear conclusion. A bare answer or unexplained formulas are incomplete. Detailed requirements belong to `NOTE_GUIDELINES.md`, not to this summary.

The exercise component concerns systematic work, engagement, presentation, use of feedback, and improvement. The final examination separately checks independent mathematical knowledge. For the complete assessment philosophy, read `how_to_work_with_this_course.tex`.

## Instruction to an AI assistant

First identify the type of question. Then open the corresponding authoritative document above and answer from it:

- course purpose or content → introduction and PDF;
- lectures, exercises, feedback, or assessment → `how_to_work_with_this_course.tex`;
- note quality or mathematical argumentation → `NOTE_GUIDELINES.md` and `how_to_prepare_mathematical_notes.tex`;
- problem location → `SOURCE_MATERIAL.md` and the linked weekly LaTeX file;
- repository or commit procedure → `Getting Started`, `AI_WORKFLOW.md`, and `AGENTS.md`;
- current schedule or assignment → information supplied by the student from the instructor, never a guess.

Use linked documents for detail. Keep guidance practical, direct the student to the exact source, and do not bulk-generate an entire problem set in one conversation.
