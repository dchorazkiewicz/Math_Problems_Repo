# Task 1 — Coin Tossing

H = Heads  
T = Tails

## One coin toss

Sample space:

Ω₁ = {H, T}

Number of outcomes:

|Ω₁| = 2

Each outcome is the result of one toss.

---

## Two coin tosses

Ω₂ = {(H,H), (H,T), (T,H), (T,T)}

Number of outcomes:

|Ω₂| = 4

Example:  
(H,T) → first toss heads, second toss tails.

---

## Three coin tosses

Ω₃ = {HHH, HHT, HTH, HTT, THH, THT, TTH, TTT}

Number of outcomes:

|Ω₃| = 8

General rule:

Number of outcomes = 2ⁿ (n = number of tosses)
